It extracts log files using the okao libraries

Usage:

Either run it without command line options to enter them manually or use them in the following order:

OKAO_extractor ParticipantID VerticalRes HorizontalRes scenarioNumber fullPathOfVideo
e.g. OKAO_extractor 1 1280 720 1 e:\eves\videos\1.mp4

spaces in path might not work
