OKAO module now accepts command line arguments for camera ID and scenario number
camera ID can be from 0 to 100
scenario can be either 1 or 2

defualt values are 0 and 1 (first available camera and scenario 1)

NOTE: use the start.bat file to run the module as it restarts the module automaticaly if it crashes for unknown reason. You can use the switches in the batch file there. (bin folder)
